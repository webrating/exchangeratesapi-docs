## Migration Guides

Since this is the initial release, no migrations are required. For users planning to integrate:

### From Other Exchange Rate APIs

<AccordionGroup>
  <Accordion title="From Fixer.io" icon="arrows-rotate">
    **Key Differences:**
    - Base currency is AUD (not EUR)
    - Australian focus with RBA official data
    - Different authentication (Bearer tokens vs query params)
    
    **Migration Steps:**
    1. Update base URL to `https://api.exchangeratesapi.com.au`
    2. Change authentication from `access_key` param to `Authorization: Bearer` header
    3. Update rate calculations for AUD base currency
  </Accordion>
  
  <Accordion title="From ExchangeRate-API.com" icon="arrows-rotate">
    **Key Differences:**
    - Official RBA data source
    - Required authentication for production use
    - More comprehensive historical data
    
    **Migration Steps:**
    1. Sign up for API key at [app.exchangeratesapi.com.au](https://app.exchangeratesapi.com.au)
    2. Update endpoints to use authentication
    3. Adjust for AUD base currency
  </Accordion>
</AccordionGroup>
